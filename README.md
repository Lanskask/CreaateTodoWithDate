# Create .todo file with current date.

Command, which create a simple todo file for notes (with current date)

Bash script, which create a .todo file. 

For example: 
  
  27.01.2017 Записки.todo

With this text inside:

```txt
- 27.01.2017 Записки
--- ✄ -------------------------- ✄ -----------------------
     
```
